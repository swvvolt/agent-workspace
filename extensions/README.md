# Extensions

Reusable extensions, integrations, plugins, MCP servers, and related resources belong here.

Add each extension in its own subdirectory with documentation, source files, and setup instructions.

## Current extensions

- [`apollo-api/`](apollo-api/) - Apollo.io API extension for Letta Code tools and people/user enrichment workflows.
- [`17track-mcp/`](17track-mcp/) - Experimental local MCP wrapper for `mcp-server-17track@1.1.3`. Not approved for broad production use until 17TRACK real-time API permission and dependency-risk review are resolved.
