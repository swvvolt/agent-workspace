# Skills

Reusable skills, prompts, workflows, and agent capabilities belong here.

Add each skill in its own subdirectory with documentation and any scripts/references needed to use it.

## Current skills

- [`17track-package-tracking/`](17track-package-tracking/) - Experimental 17TRACK package tracking skill for structured shipment lookup before carrier-page scraping. Not approved for broad production use until API authorization and dependency-risk review are resolved.
- [`firecrawl-carrier-tracking/`](firecrawl-carrier-tracking/) - Firecrawl rendered carrier-page fallback for ETA, status, current location, and delivery-proof extraction when structured sources are unavailable or insufficient.
